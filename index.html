Adding New Frames:
Support hasn't been added yet for multiple frames. However, they're easy to do:
Step 1: Draw your first frame, using this tool.
Step 2: Copy the output into a plain-text editor, like Notepad++, TextEdit, or Sublime Text.
Step 3: Add the text "^nf" to the end of your text, meaning new frame.
Step 4: Repeat for each frame, continually pasting the output after the existing text and adding "^nf" for each new frame.
